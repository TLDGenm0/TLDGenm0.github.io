# #加空格为一级标题
## ##加空格为二级标题
### 三级（一共有六级标题）

>大于符号表示 这是一段引用

有序列表用数字
1.第一步
2.第二步
3.第三步
无序列表用短横线或者星号
- 横线1
- 横线2
* 星号1
* 星号2

-空格[空格]空格任务列表
- [ ] 第一件事
- [x] 第二件事

数学公式用美元符号latex(如二次函数求根公式)
$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

`两个反引号为代码块`
```csharp
using System;
class Program {
    static void Main() {
        Console.WriteLine("Hello");
        Console.Write("hello");
    }
}
```
